# MLClusters 🚀  
**MLClusters** is an upcoming open-source machine learning clustering library, designed to enhance KMeans, KNN, and predictive clustering with advanced capabilities.  

## 🔍 About  
MLClusters is a **fork of [Khiops Enneade](https://github.com/KhiopsML/enneade)**, an older clustering tool from the Khiops suite.  
- The **original Enneade** is no longer maintained by the Khiops team.  
- It was based on an **older version of the Khiops library**.  

MLClusters aims to **modernize and enhance** Enneade by:  
- **Migrating to the latest Khiops release** for improved performance and compatibility.  
- **Refactoring and optimizing** the core clustering algorithms.  

## 🔧 Planned Features  
- 🔹 Enhanced **KMeans** with supervised pre-processing.  
- 🔹 **KNN-based clustering** for hybrid approaches.  
- 🔹 Scalable and optimized for large datasets.  
- 🔹 Improved model evaluation and reporting.  
- 🔹 Seamless integration with modern ML workflows.
